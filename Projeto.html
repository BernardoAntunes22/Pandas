Biblioteca Pandas

Neste projeto iremos explorar um pouco da biblioteca Pandas, uma das muitas bibliotecas existentes para a linguagem de programação Python. Tendo esta sido criada para manipular e analisar dados.

O Pandas tem diversas funcionalidades tais como visualização e seleção de dados, filtrar, ordenar e agrupar dados, limpar dados, entre outras. O Pandas pode ainda ser utilizado com programas remotos ao Python tais como bases de dados como a CSV, um website Json ou interligar-se com o SQL

 

 

Empresas que utilizam o Pandas

Algumas das maiores empresas do mundo trabalham ou já trabalharam com a biblioteca Pandas. Entre elas estão empresas como:

  • Facebook
  • Apple
  • Google

 

Vertentes do Pandas

Para uma utilização mais eficiente da biblioteca Pandas, é possivel interligar com diversas bibliotecas como bibliotecas de visualização como por exemplo o Matplot Lib e o scikitlearn conseguindo assim vizualizar os dados obtidos pelo Pandas tornando assim os resultados mais dinâmicos e visiveis.

 

Componente Prática do Projeto

Para compreender melhor a biblioteca Pandas, presente na linguagem python decidimos testar algumas das suas funcionalidades integrádas com um ficheiro CSV, sendo as funcionalidades testadas:

  • Mostrar
  • groupby
  • sort_values

Estas funcionalidades permitem agilizar a forma como lidamos com os dados conseguindo assim uma melhor manipulação dos mesmos, de forma a facilitar a sua análise e estudo conseguindo consequentemente melhores resultados.
