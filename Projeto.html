Biblioteca Pandas

Neste projeto iremos explorar um pouco da biblioteca Pandas, uma das muitas bibliotecas existentes para a linguagem de programação Python. Tendo esta sido criada para manipular e analisar dados.

O Pandas tem diversas funcionalidades tais como visualização e seleção de dados, filtrar, ordenar e agrupar dados, limpar dados, entre outras. O Pandas pode ainda ser utilizado com programas remotos ao Python tais como bases de dados como a CSV, um website Json ou interligar-se com o SQL
